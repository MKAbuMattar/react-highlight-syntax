# Themes

Here is how you get started with the themes:

```tsx {7} showLineNumbers
import React from 'react';
import ReactHighlightSyntax from 'react-highlight-syntax';
...
const App = () => (
  return (
    <ReactHighlightSyntax
        theme={'Base16Darcula'}
        // ...props
    />
  )
)
```

- Simply use TypeScript and choose the theme you want to use. If you want to see how the theme looks with your selected language, check out <a href="/docs/demo" className="nx-text-primary-600 nx-underline nx-decoration-from-font [text-underline-position:from-font]">Live Demo</a>.
