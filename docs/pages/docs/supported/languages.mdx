# Languages

Here is how you get started with the languages:

```tsx {7} showLineNumbers
import React from 'react';
import ReactHighlightSyntax from 'react-highlight-syntax';
...
const App = () => (
  return (
    <ReactHighlightSyntax
        language={'JavaScript'}
        // ...props
    />
  )
)
```

- Simply use TypeScript and choose the language you want to use. If you want to see how the language looks with your selected theme, check out <a href="/docs/demo" className="nx-text-primary-600 nx-underline nx-decoration-from-font [text-underline-position:from-font]">Live Demo</a>.
